# Project Documentation

This project is a news aggregator that pulls news from various sources in Lithuania. It is designed to provide users with a centralized location to access the latest news from reliable outlets.

## Features
- Aggregated news from multiple sources
- Filter news by category
- User-friendly interface

## Installation
1. Clone the repository: `git clone https://github.com/leo001-creator/lietuvos-naujienos`
2. Install dependencies: `npm install`

### Usage
Run the application with `npm start`. Visit `http://localhost:3000` to see the application in action.
